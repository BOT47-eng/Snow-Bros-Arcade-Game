# Snow-Bros-Arcade-Game
Programmers : Daniyal Ahmed Zaki & Muhammad Saad Bin Waqas 


To compile and run through commandline  :
1) Clone the directory .
2) Open the directory through terminal and run the following command : g++ -c main.cpp && g++ main.o -o sfml-app -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio && ./sfml-app
